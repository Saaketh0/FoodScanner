# FoodScanner

AI-powered nutrition tracking app with React Native frontend and Python Flask backend.

I do not have the embeddings stored on this public repo due to size, but all the code used the generate the embeddings are still here.

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
pip install -r requirements.txt
python src/server.py
```

## Features

- AI-powered food recognition using semantic similarity
- Nutrition tracking (calories, protein, carbs, fats)
- Fast cache system for common foods
- Cross-platform mobile app (iOS, Android, Web)
- Barcode Scanning
